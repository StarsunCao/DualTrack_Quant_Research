### 研究主题的相关性
- 量化交易已从规则系统演进为数据驱动的 ML 方法
- LLM 的兴起（2022-2024）为金融文本理解开辟了新的可能性
- 文献空白：传统 ML 与 LLM 方法在交易中缺乏系统性对比
- 确定不同市场条件下哪种方法更优具有实践意义

### 研究现状与发展程度
- 传统 ML 方法（LR、LSTM、梯度提升）在金融领域已有成熟应用
- 基于 LLM 的交易是新兴领域，实证证据有限
- 现有研究多聚焦单一模型或单一市场，跨模型、跨市场对比研究稀缺

- 缺乏在统一框架下公平评估 ML 与 LLM 方法的综合体系

### 研究目标与任务

  - 本文定位：这是**系统级（System-level）范式对比**而非**特征级（Feature-level）组件对比**

### 2.5 信号到仓位转换（4-5 页）
#### 2.5.1 直接信号映射
- 信号类型（买入/做空/持有/卖出）到目标权重的直接映射
- 各类信号对应的目标仓位定义

#### 2.5.2 EMA 平滑与动态衰减
- EMA 平滑：逐步仓位调整的数学原理（`smoothed = α × raw + (1-α) × prev`）
- 动态衰减：清仓信号的仓位衰减函数（`w_t = w_{t-1} × decay_rate`）
- 观望机制：hold/neutral 信号**维持当前仓位不变**（不衰减），避免过度换手
- α 和 decay_rate 参数的市场敏感性：
  - A 股：α=0.30（强平滑），decay_rate=0.70（快速衰减）
  - 美股：α=0.50（中等平滑），decay_rate=0.80（缓慢衰减）

#### 2.5.3 弱信号分类与市场差异化阈值（核心设计）
- **问题背景**：大预训练模型在交易决策中表现出显著的"保守主义倾向"——即使在明确的看多/看空判断中，输出的置信度也倾向于集中在 [0.5, 0.75] 区间，导致直接使用 `(confidence - 0.5) * 2` 线性映射时仓位过低
- **解决方案**：引入 weak_buy 与 weak_short 分类机制，低于阈值的信号被视为观望/清仓而非正常交易：
  - **weak buy**（buy + confidence < threshold）→ 视为 hold，维持当前仓位
  - **weak short**（short + confidence < threshold）→ 视为 sell，动态衰减归零
- **市场差异化配置**（网格搜索最优值）：
  - A 股（CSI300）：weak_buy_threshold = **0.60**，avg_sharpe = -0.261，avg_ret = +0.98%
    - 更严格的入场标准，减少低置信度买入导致的错误仓位
    - 对比实验：0.55 阈值 avg_ret = +0.19%，0.50 阈值 avg_ret = +0.20%
  - 美股（QQQ）：weak_buy_threshold = **0.55**，weak_short_threshold = **0.60**，avg_sharpe = +0.157，avg_ret = +5.61%
    - 美股需要更灵敏的入场信号（weak_buy=0.60 导致 avg_ret = -0.6% 至 -1.1%）
    - weak_short=0.60 相比 0.50/0.55 提供最佳风险调整后收益
  - 实验结论：A 股市场更适合严格入场（0.60），美股需要灵敏度（0.55），做空阈值统一设为 0.60
- **数学原理**：该分段函数本质上是一个带阈值门控的 Sigmoid 近似，在低置信度区施加指数级惩罚，在高置信度区保持线性激励
- **工程必要性**：若不施加此惩罚，大模型的平均仓位将不足 20%，导致回测结果失真（信号频率充足但仓位不足，夏普比率虚高但实际收益极低）

#### 2.5.4 换手率与交易成本

- EMA 平滑对换手率的抑制效果

### 3.2 ML 轨道结果（5-7 页）
#### 3.2.1 CSI300（A 股）结果
- 逻辑回归表现：+9.34%，夏普 0.1718，回撤 12.86%，换手率 0.0019
- LSTM 表现：+5.98%，夏普 0.0568，回撤 14.73%，换手率 0.0069
- LightGBM 表现：-0.02%，夏普 -0.1159，回撤 23.99%，换手率 0.0122

- 对比表与分析
- 特征重要性分析（LightGBM）

#### 3.2.2 QQQ（美股）结果
- 逻辑回归表现：+8.60%，夏普 1.0216，回撤 4.87%，换手率 0.0053
- LSTM 表现：+5.63%，夏普 0.2167，回撤 8.93%，换手率 0.0311
- LightGBM 表现：+22.42%，夏普 1.9587，回撤 8.92%，换手率 0.1006
- 集成策略表现：+11.49%，夏普 1.9916，回撤 5.35%，换手率 0.0332
- 相同模型在美股市场的对比
- 跨市场性能比较

#### 3.2.3 ML 模型对比总结
- LR 两市场夏普最高，换手率最低
- LightGBM 美股收益最高但 A 股亏损，Early Stopping 有效防止过拟合（实际树数 1-467）
- LSTM 从优化前 -40% 逆转至正收益（超参搜索 + Early Stopping + 特征筛选组合效果）
- 市场状态分析：趋势期与波动期的表现

#### 3.4.3 跨市场对比与微观市场结构影响
- A 股 vs. 美股：哪种方法在哪个市场更优
- **微观市场结构（Market Microstructure）差异的影响**：
  - T+1（A 股）与 T+0（美股）交易制度对策略执行的影响
  - 做空限制（A 股不允许做空）对策略收益上限的约束
  - 双轨智能体如何自适应这两种不同的市场物理规则：
    - A 股策略：仅做多、仓位调整更平缓（适应 T+1 隔夜风险）
    - 美股策略：可做空、仓位调整更敏捷（适应 T+0 日内变化）
- **基准漂移（Beta Drift）效应分析**：
  - 美股市场长期向上的 Beta 漂移特性对模型行为的影响
  - A 股市场的均值回归特性（无明显 Beta 漂移）对模型的影响
  - LLM 智能体在 QQQ 市场中因过度保守导致的"踏空问题"剖析
  - ML 模型在 A 股均值回归环境中的相对优势
- 新闻质量和数量在 LLM 表现中的作用
- 市场有效性对模型表现的影响

### 3.6 误差分析与局限性（2-3 页）
- ML 模型的典型失败模式与深入分析：
  - "历史未见模式"失败：当市场出现历史上从未出现过的特征组合时，ML 模型外推失效导致回撤（以具体交易日为例，展示特征空间中的异常点）
  - 过拟合噪声：训练集中拟合了噪声模式，在测试集中表现为假信号
- LLM 模型的典型失败模式与深入分析：
  - "过度解读"失败：展示一天大跌时，LLM 如何因过度解读利空新闻（如一条中性偏空的宏观新闻被解读为系统性风险信号）而错误清仓，附带 LLM 原始推理文本
  - "保守主义偏差"：LLM 在高确定性判断中仍输出中等置信度，导致仓位不足而踏空上涨行情
  - 跨语言新闻理解的局限性：中文新闻与英文新闻在 LLM 理解中的偏差
- 观察到的系统性偏差
