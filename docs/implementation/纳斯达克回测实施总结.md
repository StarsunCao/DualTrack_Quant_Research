## 架构设计亮点

### 1. 策略模式（Strategy Pattern）

```
BaseMarketStrategy (抽象基类)
    ├── AShareStrategy (A股：T+1、禁止做空、整手限制)
    └── USMarketStrategy (美股：T+0、允许做空、无限制)
```

**优势**：
- ✅ A 股回测不受影响（100% 兼容）
- ✅ 新增市场只需添加策略类（扩展性强）
- ✅ 代码复用率高（90% 代码共享）

### 2. 自动市场识别

```python
# 自动根据 symbol 选择策略和费用
run_backtest(
    ohlcv_data=data,
    target_positions=positions,
    symbol='QQQ',  # 自动选择 USMarketStrategy + SEC费
)
```

### 3. 配置驱动

```yaml
# config/backtest_config.yaml
market_config:
  us_market:
    allow_short_selling: true
    t_plus_one: false
    sec_fee_rate: 0.0000207
```
