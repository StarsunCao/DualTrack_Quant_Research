## 一、优化目标

## 二、关键优化项

### 2.1 修复内容完整性问题

**修复前**：
```
[NOT]华讯方舟:股价异动公告
```
仅包含标题，无具体内容。

**修复后**：
```
[NOT]华讯方舟:股价异动公告:重大事项 | 华讯退: 华讯方舟:股价异动公告
```
包含标题 + 内容摘要。

**修改文件**：
- `src/data/data_aligner.py` - `format_news_item()` 函数
- `src/models/llm_track/agent.py` - 优先使用 `aggregated_content`

---

### 2.2 智能筛选机制

**功能**：过滤噪音公告，保留重要信息。

**筛选规则**：
```python
notice_keywords = {
    "重大事项": ["重大事项", "重组", "并购", "收购", "投资", "中标"],
    "风险提示": ["风险提示", "警示", "退市", "亏损", "商誉减值"],
    "业绩": ["业绩预告", "业绩快报", "年报", "季报", "半年报"],
    "分红送转": ["分红", "派息", "送转", "利润分配"],
    "股权变动重要": ["实际控制人", "控股股东", "大股东"],
}
```

**效果**：
- 原始公告：423,194 条
- 筛选后：92,421 条
- 保留比例：21.8%

**新增函数**：
- `src/data/data_aligner.py` - `filter_important_notices()`

---

### 2.3 推荐配置参数

```python
max_news_per_day = 20      # 每日最多保留20条新闻
```
